Este GitHub será um espaço para registrar minha evolução, compartilhar projetos de estudo e construir meu portfólio ao longo dessa nova etapa profissional.

I am beginning my studies in Python and Artificial Intelligence, with the goal of building a solid foundation to create practical solutions, automate processes, and transform data and ideas into meaningful results.
